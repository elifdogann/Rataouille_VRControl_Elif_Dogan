## Behind the Controls: Embodying the Ratatouille Metaphor
The first video shows the player’s bodily interaction with the system. It captures how the VR controllers are used to perform movement-related gestures, directional inputs, and object-based actions. This perspective is important because the project is built around the idea that movement in VR should feel more embodied and intentional, rather than relying only on traditional button based input. By showing the player from the outside, the video makes the interaction design easier to understand and highlights the physical nature of the control scheme.

## Inside the Experience: From Physical Input to Virtual Movement
The second video focuses on the in game result of these actions. It demonstrates how the player’s controller movements are translated into navigation, steering, and interaction within the virtual world. At the same time, it gives a clearer view of the parkour structure of the environment, the thematic objects used in the game, and the feedback elements such as score, timing, and progression. This makes the demo useful not only for presenting the visual design of the project, but also for showing how the interaction system works in context.

Taken together, the videos provide a more complete picture of the project. They show that the Ratatouille inspired control metaphor is not only a conceptual starting point, but also a practical interaction model that shapes the gameplay experience. Seeing both the player’s physical actions and the virtual response makes it easier to understand the relationship between user input and system behaviour, which is one of the key concerns in VR interaction design.

The demo also reflects an important design goal of the project: finding a balance between responsiveness, clarity, and comfort. In VR, it is not enough for a system to function technically; it also needs to feel understandable and manageable for the player. For this reason, the demo helps communicate not only what the game does, but also how the interaction is intended to feel.
